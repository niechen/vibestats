# VibeStats

Measure how fast open-source projects are really being built in the AI coding era.

## Vision

VibeStats is a repo velocity intelligence platform. Instead of detecting "AI-written code," we track how engineering output accelerates over time—and whether governance keeps pace.

## Core Metrics

- **Code Acceleration**: LOC growth, commit velocity, PR merge rate, release frequency
- **Human Density**: Active contributors, output per contributor, bus factor
- **Governance Density**: Review coverage, CI presence, PR hygiene, required checks
- **Quality Guardrails**: Test ratio drift, dependency growth, generated file detection
